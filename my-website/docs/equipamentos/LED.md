# LED 

Os sinalizadores são equipamentos importantes para a identificação de passagem de energia em um aparelho, podendo variar em sinais luminosos, sonoros e vibracionais. Podem ser utilizados tanto para verificação de um motor acionado, quanto para uma irregularidade na instalação, um exemplo deles é o LED que utiliza com sinalização sinais luminosos.
