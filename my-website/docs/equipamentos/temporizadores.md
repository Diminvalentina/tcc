# Temporizadores

O temporizador pode ser utilizado para ativar e desativar dispositivos conforme a necessidade do programador. O temporizador disponível no laboratório 4 do IFPR - Câmpus Assis Chateaubriand, é do tipo comutador que redireciona a corrente de uma entrada para outra quando o tempo estipulado é excedido
